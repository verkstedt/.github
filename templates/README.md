# Templates

Assortment of templates of files.

## `dependabot.yaml`

[🔍 Repositories using this template](https://github.com/search?q=-is:archived+path:.github/dependabot.y*ml+"https://github.com/verkstedt/.github/blob/main/templates/dependabot.yaml"&type=code)

In addition to using this template, you should also set somebodies as code owners of files that dependabot updates in PR to make sure people get assigned to them:

```
# Make sure dependabot PRs get reviewers assigned
package-lock.json                   @YOU
yarn-lock                           @YOU
pnpm-lock.yaml                      @YOU
Dockerfile                          @YOU
docker-compose.*                    @YOU
/.devcontainer/devcontainer.json    @YOU
/.github/workflows/                 @YOU
```
